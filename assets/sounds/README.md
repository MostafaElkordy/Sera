# Audio Files Guide

This directory contains the sound files for the SERA Emergency App.

## Required Sound Files:

1. **beep.wav** - A simple beep sound that plays during the countdown (each second)
   - Duration: ~200ms
   - Frequency: 800Hz
   - Format: WAV, 44100Hz sample rate

2. **confirm.wav** - A pleasant notification sound when SOS alert is confirmed
   - Duration: ~500ms
   - Format: WAV, 44100Hz sample rate
   - Should be a double beep or ascending tone

3. **cancel.wav** - A gentle notification sound when the SOS is cancelled
   - Duration: ~400ms
   - Format: WAV, 44100Hz sample rate
   - Should be a single beep or descending tone

## How to add the sounds:

You can use free online sound generators or download from free sound libraries:
- https://freesound.org/
- https://www.zapsplat.com/
- https://www.soundly.com/

Or generate using:
- Audacity (free, open-source)
- Online tone generators

Place the .wav or .mp3 files in this directory.
